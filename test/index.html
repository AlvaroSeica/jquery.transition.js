A animated menu built with CSS3 Transitions that works in IE?

  • Yes
  • Now
  • It is

How to use it?

  1. Declare an animation using CSS3 Transitions and the :hover pseudo-class.
  2. Include jQuery and jquery.hoverTransition.js in your page.
  3. Test your page in IE.

It is recommended to use this page as a start point and have a look at the full documentation of the jQuery jQuery Transition Project.
